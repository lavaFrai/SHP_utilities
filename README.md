# SHP_utilities

### How to use:

#### Linux
```shell
git clone "https://github.com/lavaFrai/SHP_utilities"
cd SHP_utilities
python3 main.py
```


#### Windows
```shell
git clone "https://github.com/lavaFrai/SHP_utilities"
cd SHP_utilities
python main.py
```

That's all, type this in shel and enjoy the program!
